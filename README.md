# Exo : API Rick and Morty

Réaliser un programme permettant d'exploiter les fonctionnalités de l'API Rick and Morty disponible à cette adresse :

```text
https://rickandmortyapi.com/
```

Le programme devra permettre plusieurs fonctionnalités :

* Permettre de chercher un personnage via son **nom**
* Permettre de chercher un personnage via son **identifiant**
* Permettre de visualiser les **informations principales** du personnage (son **nom**, son **espèce**, son **genre**, son **statut**, son **origine**, sa **dernière localisation connue**, et l'**image** du personnage)
* Il devra être possible de passer directement au personnage **suivant ou précédent** dans la base de données (aller du #27 au #28 par exemple). Cette fonctionnalité devra vérifier si l'on est déjà au **premier ou au dernier personnage** afin d’éviter les erreurs

---

### BONUS (pour les tryhards)

* Il devra être possible d’accéder à la **liste des épisodes**, d’en sélectionner un, pour ensuite afficher **tous les personnages présents dans cet épisode**
* Il devra être possible, dans la fiche personnage, d’accéder à la **liste complète des épisodes** dans lesquels il apparaît
* Il devra être possible de cliquer sur **l’espèce**, le **genre** ou le **statut** du personnage pour obtenir un **listing de tous les personnages** partageant cette même caractéristique
